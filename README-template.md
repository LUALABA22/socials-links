### What I learned

During thisproject, i learned how to use the `:focus ` pseudo-class on links to improve accessibility for users who navigate using the keyboard.
This makes the active element visible when using the "Tab" key, improving user experience and following accessibility best practices.

Example:

```css
.card a:focus {
 background-color: hsl(75, 94%, 57%);
  color: black;
  outline: 2px solid black;
}
```
I also strengthened my understanding of Flexbox to align and center element officiently.
